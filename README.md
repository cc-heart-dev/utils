# @cc-heart/utils

[Docs](https://cc-hearts.github.io/utils/)

A library of JavaScript tools

## Install

```shell
npm install @cc-heart/utils
```

## Usage

```js
import { capitalize } from '@cc-heart/utils'

capitalize('string') // String
```
